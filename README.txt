# DEV INSTALLATION

1) `npm install`
2) `npm install -g node-dev`
2) `brew update && brew install rabbitmq`
2) `npm install -g supervisor`
3) `npm install -g grunt-cli`
4) `grunt install`
5) `grunt watch` to automatically build and concat frontend JS/LESSin
/dev/js/* and /dev/less/*.
6) `grunt s` to start the server
7) `/usr/local/sbin/rabbitmq-server`
8) `node-dev worker/worker.coffee`
